## Intruções de Uso

Este repositório está remotamente conectado tanto ao github quando ao cpnael, na hora de realizar o `git push` atente-se à subir as atualizações para os dois ambientes:

**CPanel**

```
git push cpanel main
```

**Github**

```
git push origin main
```
